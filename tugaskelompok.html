Pengertian Tag, Elemen, dan Atribut pada HTML.

Tag, Elemen, dan Atribut merupakan dasar dari terbentuknya sebuah halaman web. HTML (Hyper Text MarkUp Language) adalah teks yang digunakan untuk menyusun tampilan web dan sudah memiliki syntax. Tag adalah sebuah tanda yang digunakan HTML untuk memberitahukan kepada web untuk apa fungsi dari sebuah teks. Element merupakan isi dari tag yang berada di antara tag pembuka dan tag penutup. Atribut merupakan informasi tambahan yang diberikan kepada tag. Informasi yang dimaksud bisa berupa perintah untuk warna dari teks, ukuran huruf teks, menentukan posisi teks, dan masih banyak lagi. Setiap atribut memiliki pasangan berupa nama dan nilai atau ditulis dengan name ="value".

Tag- tag Dasar HTML

<br> = Pembuatan garis baru
<hr> = Pembuatan garis horizontal
<pre> = Menampilkan teks dengan mempertahankan enter dan spasi.
<div> = Pengelompokan sejumlah elemen
<b> </b> = Bold mempertebal tulisan teks
<i> </i> = Italic menampilkan teks dalam bentuk miring
<u> </u> = Underline menampilkan teks disertai dengan garis bawah

Apa itu HTML?
HTML (HyperText Markup Language) Bahasa yang digunakan untuk membuat sebuah halaman Web, menampilkan berbagai informasi di dalam sebuah penjelajah web internet dan dapat dipulikasikan. Dan diikuti oleh aturan XML.

Apa itu Tag ?
Tag adalah Suatu tanda pengenal pada html yang terdiri dari 2 bagian yaitu tag pembuka yang ditandai dengan "<" dan tag penutup yang ditandai dengan "/>".
Contoh : <Head> dan <Body>.

Apa itu Elemen ?
Elemen adalah Isi dari sebuah Tag yang berada diantara tag pembuka dan tag penutup.
Contoh : <p> adalah sebuah paragraf </p>.

Apa itu Attribute ?
Attribute adalah Infomasi tambahan yang diberikan kepada Tag.
Contoh : Warna, Ukuran Teks, Jenis Teks, dsb.

Tag-Tag Dasar HTML 

 <!DOCTYPE> -> Tag untuk menentukan tipe dokumen. 
			
 <html>     	-> Tag untuk membuat sebuah dokumen HTML. 
			
 <title>    	-> Tag untuk membuat judul dari sebuah halaman.
			
 <body>     	-> Tag untuk membuat tubuh dari sebuah halaman.
			
 <h1>to<h6> -> Tag untuk membuah Heading.
			
 <p>        	-> Tag untuk membuat paragraf.
			
 <br>       	-> Tag Membuat baris baru.
			
 <hr>       	-> Tag untuk membuat perubahan dasar kata didalam isi.
			
 <!-->      	-> Tag untuk membuat komentar.

Bagian-bagian yang terdapat pada Body :
<Base> → Membuka link yang terdapat dalam dokumen
<Link> → Menghubungkan dokumen Eksternal ke resource (Link Global).
<Meta> → Memberikan tambahan informasi ke dalam halaman web.
